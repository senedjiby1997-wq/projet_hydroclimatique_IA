# Projet Hydroclimatique IA

Ce projet prédit la pluviométrie à partir de données météo historiques pour aider à la planification agricole.

## Structure
- `src/` : scripts Python pour traitement des données et modèle
- `data/` : données météo
- `model/` : modèles sauvegardés
- `requirements.txt` : librairies nécessaires
